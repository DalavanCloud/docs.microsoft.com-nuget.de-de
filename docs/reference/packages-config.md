## <a name="examples"></a><span data-ttu-id="0a732-141">Beispiele</span><span class="sxs-lookup"><span data-stu-id="0a732-141">Examples</span></span>

```xml

  <package id="NLog" version="4.3.10" targetFramework="net46" />
</packages>
```
