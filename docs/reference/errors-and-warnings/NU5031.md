# <a name="nuget-error-nu5031"></a>NuGet-Fehler NU5031
<pre>The license file 'LICENSE.txt' has an invalid extension. Valid options are .txt, .md or none.</pre>

### <a name="issue"></a>Problem

Nur die aufgelisteten Erweiterungen sind zulässig.

### <a name="solution"></a>Lösung

Verwenden Sie eine aus der Liste der zulässigen Erweiterungen.
